# Frågeställning 1 

Vilken krypteringsalgoritm (AES vs RSA) är mest effektiv för att säkra dataöverföring utan att orsaka hög latens? 

# Metod 1 

Genom implementering av ett skript (i Python) kan variablerna isoleras för att säkerställa ett tillförlitligt resultat. Hårdvaran och filinnehållet hålls konstanta under hela testprocessen medan endast algoritmen och filstorleken manipuleras. Varje enskilt test utförs 10 gånger för att generera ett genomsnittligt värde och minimera felmarginaler orsakade av bakgrundsprocesser i datorn.
